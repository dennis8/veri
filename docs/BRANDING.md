# veri Branding Guidelines

## Name & Tagline
- **Name:** veri
- **Tagline:** "Ultra-fast, pytest-compatible, impact-aware test runner"
- **Emoji:** ⚡ (lightning bolt - represents speed)
- **Color:** `#FF6B35` (energetic orange - represents speed and reliability)

## Usage
- Always lowercase: `veri`
- CLI banner should include the lightning emoji
- Documentation headers can use the tagline

## Visual Identity
- Primary color: `#FF6B35` (Fast Orange)
- Secondary color: `#4ECDC4` (Trust Teal)
- Accent color: `#FFE66D` (Highlight Yellow)
